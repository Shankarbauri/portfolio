# 🌐 Shankar Kumar Bauri — Portfolio

A modern, responsive personal portfolio website built using HTML, CSS, and JavaScript to showcase my skills, projects, and professional profile in a clean and interactive way.

---

## 📖 About

This portfolio is designed to present my work as a developer with a strong focus on UI/UX and smooth user experience. It combines structured layout with visually appealing animations to create an engaging interface for visitors. The project reflects my ability to build responsive and user-friendly web applications using core web technologies.

---

## ✨ Features

* Animated gradient hero section with floating particles
* Smooth scroll and reveal-on-scroll animations
* Interactive project cards with hover effects
* Animated skill progress bars
* Fully responsive design (mobile-first)
* Contact form with success message handling
* Clean and modern UI/UX design

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3 / Tailwind CSS** – Styling & animations
* **JavaScript (Vanilla)** – Interactivity
